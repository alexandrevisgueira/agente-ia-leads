# 🤖 Agente de IA: Qualificador de Leads de Saúde

Sistema inteligente que utiliza a API do Gemini para processar listas de leads em massa, classificando-os por impacto estratégico e gerando relatórios individuais de abordagem.

## 🚀 Funcionalidades
- **Qualificação Automática:** Classifica leads como Alto, Médio ou Baixo impacto baseado em critérios de negócio.
- **Relatórios Premium:** Gera automaticamente arquivos `.txt` com análise de dor e sugestão de abordagem consultiva.
- **Processamento em Escala:** Leitura de arquivos CSV com barra de progresso em tempo real.

## 🛠️ Tecnologias
- Python 3.14
- Google Gemini API (Model: Flash-Latest)
- Bibliotecas: `requests`, `python-dotenv`, `tqdm`
